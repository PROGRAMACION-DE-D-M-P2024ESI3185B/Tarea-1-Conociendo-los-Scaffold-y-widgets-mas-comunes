Material App Bar:
Esta aplicación es una interfaz donde se podrá ver información de usuario que es el nombre, trabajo e información importante como la dirección y número de teléfono.
Dentro de esta aplicación se encuentran 4 botones que cambiaran de color si presionas y dispararan snacksbars diferentes dependiendo de que sea disparara un mensaje diferente.
